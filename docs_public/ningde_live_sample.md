# 活样本：一笔亏损，如何变成系统里的一条新规则

> 这是明仓"研究决策闭环"跑通一次的**真实纸上交易记录**（非真金白银）。
> 它演示的不是"AI 选股多准"，而是 **判断 → 结果 → 复盘 → 记忆 → 改进** 这条完整闭环。
> 标的：**300750 宁德时代**，区间 2026-05-14 ~ 05-25。

---

### ① 研究 ｜ 为什么值得进 · `L2 论题 / L3 信号`

```
进口判断：固态电池 — 全固态 2026 量产，储能出海双轮驱动
入场信号 05-14：综合 +21（买入）｜ 量化 +1.1 · 技术 +26.4 · 情感 +0.65
执行：开盘价 449.38 买入，仓位 20%，止损 395.57 / 止盈 493.69
```

### ② 持仓 ｜ 信号开始转弱，但规则没让我出场 · `L3 持仓纪律`

```
05-19  信号 -7.8（观望）  技术转弱       浮亏约 -7%   止损 395.57 未破（盘中最低 406.80）
05-22  信号 +4.9（可关注） 反复拉锯
05-25  信号 -15.5（观望）  技术 -40.8 极弱  ATR 扩大 → 止损线反向下移至 373.13
```
> ⚠️ 关键：价格**从未跌破机械止损线**，机械止损在这里**帮不上忙**——信号早已极弱，
> 但当时**没有一条"信号反转就退出"的规则**，只能眼睁睁硬扛。

### ③ 平仓 ｜ 人工主动认亏 · `L3 → L4`

```
05-25  人工判断趋势不支持继续持有，主动平仓 @ 411.28
       净亏损 −8.48%（仓位贡献 −1.70%）
```

### ④ 复盘归因 ｜ 写下根因，不甩锅给运气 · `L4 复盘案卷`

> **根因结论（写进 test1 复盘）：**
> 宁德这笔亏损 **不是机械止损的问题，而是缺少"信号反转退出规则"** ——
> 信号 05-19 就转弱了，系统却没有规则把人请出场。

### ⑤ 记忆 → 改进 ｜ 教训长成下一版的规则 · `L0 记忆 → 下一轮`

```
记忆候选（人工确认后晋升为可信记忆）：
  "信号连续转弱应触发退出，不能只依赖机械止损线。"
        │
        ▼
测试2 v2 新规则：新增「信号反转退出」+ 更大 ATR 倍数
```

## 这条闭环，每一步都落在系统里

| 闭环的一步 | 落在明仓哪里 |
|---|---|
| 出信号 | 每日信号 |
| 记录这次判断的理由 | ✅ 研究案卷 |
| 出结果后做归因 | ✅ 复盘案卷 |
| 把教训沉淀成记忆 | ✅ 记忆候选 → 可信记忆 |
| 教训改进下一版规则 | ✅ 测试2 新规则 |

> **亏的 8.48% 没白亏，它变成了系统记忆里的一条规则。**
> 这就是"会成长的研究系统"的意义所在。
